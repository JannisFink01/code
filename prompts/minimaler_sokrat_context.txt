'## Fachlicher Kontext: Z-Diode und Spannungsstabilisierung


### 1. Diodengrundlagen
- Am pn-Übergang entsteht eine Raumladungszone (Sperrschicht).
- In Durchlassrichtung: Raumladungszone wird kleiner, ab Schwellspannung (~0,6–0,7 V bei Si) fließt Strom.
- In Sperrrichtung: Raumladungszone wird größer, es fließt nur ein sehr geringer Sperrstrom.

### 2. Z-Diode (Zener-Diode)
- Betrieb in Sperrrichtung; Durchbruchspannung UZ ist technologisch einstellbar.
- Beispielbauteil: BZX79-B12 (Nexperia); UZ = 11,80–12,20 V bei IZ,test = 5 mA.
- Differenzieller Widerstand: rD,typ = 10 Ω (bei 5 mA), rD,typ = 50 Ω (bei 1 mA).
- Grenzwerte: IF = 250 mA, Ptot = 500 mW (bei 50 °C).
- Kennlinie folgt einer e-Funktion; für Berechnungen wird linearisiert.

### 3. Ersatzschaltbild
- Durchlassbereich: idealer Schalter + Durchlassspannung UD0 + differenzieller Widerstand rD.
  Gleichung: UD = UD0 + rD · ID
- Sperrbereich (Z-Diode): idealer Schalter + Zenerspannung UZ0 + differenzieller Widerstand rZ.
  Gleichung: UZ = UZ0 + rZ · IZ
- Arbeitsbereich: IZmin ≤ IZ ≤ IZmax (IZmin ≈ 0,1 · IZmax; IZmax = Ptot / UZ0)

### 4. Einfache Stabilisierungsschaltung
- Grundschaltung: Spannungsquelle UE → Vorwiderstand RV → Parallelschaltung Z-Diode und Last RA.
- Knotenregel: IE = IZ + IA
- Maschenregel: UE = URV + UZ
- Line regulation: dUA/dUE = rZ / (rZ + RV) → je kleiner rZ, desto besser.
- Load regulation: dUA/dIA = −(rZ · RV) / (rZ + RV) → je kleiner rZ, desto besser.
- Ziel: rZ möglichst klein → Arbeitsbereich genau einhalten.

### 5. Längsregler (Transistor + Z-Diode)
- Transistor (Kollektorschaltung/Emitterfolger) ersetzt den Ausgangswiderstand RA.
- Ausgangsspannung: UA = UZ − UBE
- Kompensation von UBE durch zusätzliche Diode D2 möglich.
- Vorwiderstand: RV = (UE − UZ) / (IB + IZmin)
- Basisstrom: IB = (UZ − UBE) / ((1 + β) · RA)
- Vorteil gegenüber einfacher Schaltung: Transistor übernimmt den Laststrom, Z-Diode wird entlastet.
